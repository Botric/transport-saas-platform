# Passenger App

This directory will contain the Passenger App.

## Build Options

Per the product spec, the passenger app can be built as:
- Android APK (recommended after Driver App)
- React Native app (iOS + Android)
- Progressive Web App (PWA)

## Status

Planned — Phase 4 of the roadmap. Work begins after the Driver App, Backend API, and Web Portal MVPs are complete.

## Planned Features

- Passenger account login/register
- Region and route selection with saved preferences
- Live vehicle location on map
- ETA to selected stop
- Capacity display
- Ticket purchase and free claim
- Active and previous orders
- QR/barcode ticket display
